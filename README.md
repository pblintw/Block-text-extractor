# 區塊文字擷取器 (Block Text Extractor)

一個簡潔實用的 Chrome 擴充功能，讓你輕鬆選擇網頁上的任意區塊，並將其內容複製為純文字或 Markdown 格式。

![Preview](preview.png)

## ✨ 功能特色

- **視覺化選擇**：滑鼠移動時即時高亮顯示可選區塊
- **雙格式支援**：
  - **純文字**：乾淨的文字內容，去除所有格式
  - **Markdown**：保留標題、列表、連結、粗體、斜體等格式
- **自動複製**：擷取後自動複製到剪貼簿
- **快捷操作**：按 ESC 隨時取消選擇

## 📦 安裝方式

### 方法一：載入未封裝的擴充功能（開發者模式）

1. 下載或解壓縮此擴充功能資料夾
2. 開啟 Chrome 瀏覽器，前往 `chrome://extensions/`
3. 開啟右上角的「**開發人員模式**」
4. 點擊「**載入未封裝項目**」
5. 選擇此擴充功能的資料夾
6. 完成！擴充功能圖示會出現在工具列中

### 方法二：打包擴充功能

1. 前往 `chrome://extensions/`
2. 開啟「開發人員模式」
3. 點擊「打包擴充功能」
4. 選擇此資料夾作為根目錄
5. 生成的 `.crx` 檔案可以分享給其他人安裝

## 🚀 使用方式

1. **點擊擴充功能圖示**：開啟控制面板

2. **選擇輸出格式**：
   - 「純文字」：只保留文字內容
   - 「Markdown」：保留格式結構

3. **點擊「開始選擇區塊」**：
   - 游標會變成十字形
   - 滑鼠移動時，區塊會以紫色邊框高亮

4. **點擊目標區塊**：
   - 文字會自動擷取並複製到剪貼簿
   - 結果顯示在控制面板中

5. **按 ESC 取消**：隨時可以取消選擇模式

## 📝 Markdown 轉換支援

| HTML 元素 | Markdown 輸出 |
|-----------|---------------|
| `<h1>` - `<h6>` | `#` - `######` |
| `<strong>`, `<b>` | `**粗體**` |
| `<em>`, `<i>` | `*斜體*` |
| `<del>`, `<s>` | `~~刪除線~~` |
| `<code>` | `` `程式碼` `` |
| `<pre>` | 程式碼區塊 |
| `<a>` | `[連結](URL)` |
| `<img>` | `![alt](src)` |
| `<ul>`, `<ol>` | 無序/有序列表 |
| `<blockquote>` | `> 引用` |
| `<table>` | Markdown 表格 |
| `<hr>` | `---` |

## 🔧 技術細節

- **Manifest Version**: 3
- **權限需求**：
  - `activeTab`：存取當前分頁
  - `clipboardWrite`：寫入剪貼簿

## 📁 檔案結構

```
chrome-extension/
├── manifest.json        # 擴充功能設定
├── popup.html          # 彈出視窗 HTML
├── popup.css           # 彈出視窗樣式
├── popup.js            # 彈出視窗邏輯
├── content.js          # 內容腳本（區塊選擇與文字擷取）
├── content-styles.css  # 內容腳本樣式
├── icons/              # 擴充功能圖示
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md           # 說明文件
```

## 🐛 已知限制

- 某些使用 Shadow DOM 的網站可能無法正確選取內部元素
- 部分動態載入的內容可能需要等待載入完成
- iframe 內的內容無法直接選取
